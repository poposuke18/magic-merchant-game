# 魔術商人（Magic Merchant Game）

## プロジェクト概要
プレイヤーは魔術書を売る商人となり、人間側と魔物側の勢力がどちらも滅亡しないようにバランスを保ちながら、最大限の利益を上げることを目指すゲーム。
